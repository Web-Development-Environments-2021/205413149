Ans



Web 1.0: The first generation of the web. created by Tim Berners Lee in 1989 and lasted until 2005.
web 1.0 was a read-only version of the internet and had almost no user-web interaction.

Web 2.0: The second version of the web which had read-write interaction. Web 2.0 added the abillity
to share content and information of users (social media for example). it changed the way we interact
with each other and with the internet.

Web 3.0: This version includes the ability of semantic text tagging. Offers user oriented content.
More intelligent and sophisticated version of the web. to share content and information of users
(social media for example). it changed the way we interact.

For example, if a user A types the next query in google search engine: "interesting trips abroad",
in Web 1.0 he would get the same results as user B. On the other hand,
in Web 3.0 user A will recieve relevant results for user A only.
as mentioned above, web 3.0 has the ability to present user-oriented content.
